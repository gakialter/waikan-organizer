# Waikan Organizer (考研外刊英语整理工具)

这个脚本（`organize_waikan.py`）专为考研英语学习者设计。它可以将原本散乱在不同日期、不同文件夹下的 PDF 和 Word 学习资料，按照日期排列、难度评级、以及预设的阅读顺序重新归档重命名。

## 主要功能 (Key Features)

- 📈 **难度自动提取**: 自动分析 `精读.pdf` 甚至后备的 `杂志排版.pdf` 首页内容，智能提取文章的星级难度（如 ★★★☆☆）。
- ⚡ **增量处理与去重**: 在第二次甚至后续的无数次运行中仅仅处理新文件。遇到不同来源的重复同名文件，自动比较修改时间保留最新下载的版本。
- 🗓️ **日期零填充排序**: 自动将 `3.2` 等参差不齐的日期规范格式化为 `03.02`，确保在 Windows 资源管理器里严格按时间线顺滑排列，专治强迫症。
- 🤖 **自动纠错与巡检**: 在整理完成后，自动输出一份“缺漏清单”，告诉你各天缺少了什么核心文件，防止学习资料遗漏。
- 🛡️ **高健壮性**: 对各种奇怪的扩展名、特殊的每周合并篇章、甚至中文加Emoji环境都具有极强的兼容和自动跳过机制。

## 使用方法 (Usage)

1. 安装非常基础的依赖库（只需要用来解析PDF难度）：
   ```bash
   pip install pdfplumber
   ```
   
2. 把 `organize_waikan.py` 放到你的包含各个考研外刊杂乱分文件夹的根目录。

3. 通过控制台/CMD/Powershell 运行脚本：
   ```bash
   python organize_waikan.py
   ```

4. 脚本会自动创建一个名叫 `整理后外刊文档` 的全新文件夹，里面会有极致干爽的按星级分类和日期命名的结构！
